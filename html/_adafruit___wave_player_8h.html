Detailed Description

This is part of Adafruit's WavePlayer Arduino library. It's really more a WAV helper than a player, providing some of the lower-level essentials. The examples show how to turn this into a working player.

Written by Phil "PaintYourDragon" Burgess for Adafruit Industries.
